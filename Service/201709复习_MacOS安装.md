1. 安装IDEA的Ut高级版
2. 在激活处输入http://idea.iteblog.com/key.php并点击Active激活成功;
3. 安装Tomcat
	
	```
	1. 到官网下载
	2. cd到tomcat/bin目录
	3. sudo sh startup.sh启动
	
	注:
	sudo sh shutdown.sh是关闭;
	sudo sh catalina.sh是查看当时tomcat版本;
	```

4. 到IDEA的RUN Config中设置Tomcat
	
	```
	1. 点+号;添加Tomcat目录;
	```
	
5. 安装JDK

	```
	1. 到官网下载
	2. 安装
	```
